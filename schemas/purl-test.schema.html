Test type

Type: enum (of string)

The type of this test like 'build' or 'parse'.

Must be one of:

  • "build" :

    A PURL building test from decoded components to a canonical PURL string.

  • "parse" :

    A PURL building test from decoded components to a canonical PURL string.

  • "roundtrip" :

    A PURL roundtrip test, parsing then building back a PURL from a canonical string input.
